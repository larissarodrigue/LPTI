1- Criar atividade (acadêmica)
1.1- Para criar uma atividade o usuário deve escolher a opção: atividades progamadas
1.2- Feito isso o usuário poderá escolher a opção de adicionar nova atividade
1.3- Para criar uma nova atividade o usuário deve fornecer o nome, data, a turma que irá realizar, nota e alguma possível observação.
1.4- Após preenchida as informações, basta salvar a nova atividade
 OBS: Essas atividades são para fins acadêmicos

1- Criar atividade (pessoal) curso normal
1.1- Para criar uma atividade o usuário deve escolher a opção atividades progamadas
1.2- Feito isso o usuário poderá escolher a opção de adicionar nova atividade
1.3- Como a atividade é pessoal o usário pode escrever qualquer coisa para identifica-lá 
1.4- 1.4- Após preenchida as informações, basta salvar a nova atividade

2- Editar atividade 
2.1- Para acessar uma atividade já criada o usuário deve escolher a opção: atividades progamadas
2.2- Feiro isso, é preciso clicar em editar atividade
2.3- O usuário deve mudar os dados desejados
2.4- Após modificados o sistema irá salvar as novas informações
